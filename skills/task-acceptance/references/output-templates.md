# Output Templates

> 任务收尾技能 output 模板参考

---

## 验收报告模板

```markdown
## 验收报告

- [x] <验收项1> - 验证方法：<具体执行了什么>
- [x] <验收项2> - 验证方法：<具体执行了什么>
- [ ] <验收项3> - 未通过原因：<如实记录>
```

---

## 任务总结模板

```markdown
## 任务总结

### 实现方案

<简要描述采用了什么方案、技术路径、关键实现点>

### 执行结果

<描述实际完成情况，与目标的对比>

### 产出清单

- <产出物1>：<路径或说明>
- <产出物2>：<路径或说明>
```

## Reference 章节模板

```markdown
## Reference

| 文档 | 功能 | 路径 |
|------|------|------|
| <文档名> | <描述文档用途> | <相对路径或绝对路径> |
```

## lessons-learned.md 模板

```markdown
# <任务ID> <任务名称> - 复盘总结

## 问题与对策

| 问题 | 对策 | 结果 |
|------|------|------|
| <问题描述> | <采用的解决方案> | <效果如何> |

## 关键决策

| 决策 | 背景 | 选择 | 理由 |
|------|------|------|------|
| <决策内容> | <当时情况> | <选择了什么方案> | <为什么这样选> |

## 经验教训

1. <经验1>
2. <经验2>
```

# Harvey 自定义设计系统

> 本文件是 Harvey 自定义设计系统的单一真相来源，灵感源自咨询行业视觉风格（深海军蓝配色体系）。
> 与 `rb-style.md`（Roland Berger 官方品牌标准）区分开：本文件不追求与官方 RB 品牌完全一致。
> 任何需要生成样式配置的 skill（doc-forge、mermaid-diagram 等）应先读取本文件，再按目标格式推导具体配置。

---

## 1. 品牌概述

Roland Berger 风格的核心特征：
- **专业沉稳**：深海军蓝为主色，传递权威感与信任感
- **层次清晰**：通过色彩深浅表达信息层级，而非大量装饰
- **克制简洁**：无多余边框、无渐变、无阴影；用空白和对齐传递结构
- **中西文混排友好**：中英文字体分别指定，字号体系统一

| 场景 | 中文字体 | 英文字体 | 说明 |

## 5. 组件规则

### 5.1 标题装饰

| 级别 | 装饰方式 |
|------|---------|
| H1 | 下方细线 `#00205B`，2–2.5px |
| H2 | 下方细线 `#003E96`，1.5px |
| H3 | 左侧色条 `#C5002B`，3px，内缩 8pt |
| H4 | 无装饰，uppercase + letter-spacing |

### 5.2 表格（水平线风格）

- **无竖线，无单元格边框**
- 表头上方：粗横线（1.5px / sz=12）
- 表头下方：细横线（0.75px / sz=6）
- 末行下方：细横线
- 表头：加粗，无背景色
- 数据行：无背景色，无斑马纹

### 5.3 代码块

- 背景色 `#F4F6FA`，左侧装饰线 `#002B6C`，3px
- 圆角 3px（右侧），左侧直角
- 字体：`JetBrains Mono` / `Courier New`，9–10pt

### 5.4 引用块

- 背景色 `#FDF5F6`，左侧装饰线 `#C5002B`，3.5px
- 斜体，颜色 `#4A5568`，字号略小于正文

### 5.5 分隔线

- 颜色 `#BFC8DC`，1–1.5px，无装饰

## 6. 格式推导指南

读取本文件后，按以下规则生成目标格式配置：

### → DOCX（default-style.json）

- `body.font` = `PingFang SC`，`body.font_en` = `Arial`
- `headings.h1.color` = `000000` 或 `00205B`（取深色）
- `table.border_mode` = `"mckinsey"`，`table.rule_color` = `"000000"`
- `table.header_bg_color` = `null`
- 字号、间距参照第 3–4 节

### → PDF（default.css）

- `font-family` 栈以 `PingFang SC` 开头
- `h1` 用 `border-bottom: 2.5px solid #002B6C`
- `h2` 用 `border-bottom: 1.5px solid #002B6C`
- `h3` 用 `border-left: 3px solid #C5002B`
- `table` 水平线规则：`thead tr { border-top: 1.5px; border-bottom: 0.75px }`，`td/th { border: none }`
- 颜色参照第 2.1 节

### → Mermaid style 块

- `subgraph` 三层色：`#00205B` → `#003E96` → `#1E5C9E`
- 节点三层色：`#0A2E7A` → `#0050B8` → `#2A6EAE`
- 深色节点一律加 `color:#fff`
- `sequenceDiagram` / `gantt` / `timeline` 不着色，使用库默认
- 语义色参照第 2.3 节

# Roland Berger 官方品牌标准

> 本文件记录 Roland Berger 官方品牌视觉语言，基于 rolandberger.com 公开发布的 CSS 样式与视觉资产整理。
> 与 `custom-style.md`（Harvey 自定义设计系统）区分开：本文件追求还原原汁原味的 RB 品牌风格。

---

## 1. 品牌概述

Roland Berger 官方品牌的核心特征：
- **黑白底色为主**：页面背景为白色，主文字为黑色，大量使用留白
- **品牌色作为强调**：亮黄色 `#f6f600` 为第一强调色（CTA、标题下划线、高亮）
- **青色为辅助强调**：`#00aac9` 用于次级 CTA、图标、装饰元素
- **高对比、现代感**：无渐变，无阴影；用颜色块、粗字重传递层级
- **专有字体主导**：正文使用 `RBDesign` 专有字体家族（不开源）

## 2. 色彩体系

### 2.1 主色板

| 角色 | 色值 | 用途 |
|------|------|------|
| Black | `#000000` | 主文字、标题、主要线条 |
| White | `#ffffff` | 页面背景、深色区块内文字 |
| Brand Yellow | `#f6f600` | 第一强调色：CTA 按钮底色、重要标题下划线、高亮块 |
| Brand Cyan | `#00aac9` | 第二强调色：次级 CTA、图标、信息标签 |
| Brand Pink | `#e6006e` | 第三强调色：品牌特殊场景使用（如特定子品牌） |
| Medium Blue | `#156c9c` | 正文链接色、深色背景下的次级按钮 |

### 2.2 功能色

| 角色 | 色值 | 用途 |
|------|------|------|
| CTA Red | `#ff3532` | 主要行动按钮背景（"Download", "Contact us"） |
| Dark Teal | `#003b46` | 导航栏背景、深色区块底色（hero 区、卡片背景）；CSS 类名 `bg-darkgreen` |
| Subtle Gray | `#f5f5f5` | 卡片背景、交替行背景 |
| Border Gray | `#e0e0e0` | 分隔线、边框 |
| Body Gray | `#333333` | 正文辅助文字 |

### 2.3 图表/数据可视化辅助色

官网图表中观察到的配色（非官方命名，按使用频率排序）：

| 角色 | 色值 |
|------|------|

| 数据色 2 | `#f6f600`（yellow） |

## 3. 字体体系

### 3.1 官方字体（不开源，需授权使用）

| 场景 | 字体 | 说明 |
|------|------|------|
| 正文 / 标题 | `RBDesign` | 专有无衬线字体，有 Light / Regular / Medium / Bold 等字重 |
| 中文正文 | `RBDesign Chs` | 中文版专有字体 |
| 编辑专题 / 报告引言 | `Arnhem` | 衬线字体，用于大字号引言、封面强调文字 |

### 3.2 降级字体栈（无授权时使用）

**英文：** `"Helvetica Neue", "Arial", "sans-serif"`

**中文：** `"PingFang SC", "STHeiti", "Microsoft YaHei", "sans-serif"`

**混排：** `"RBDesign", "PingFang SC", "Helvetica Neue", "Arial", "STHeiti", "Microsoft YaHei", sans-serif`

### 3.3 字号体系（观察自官网）

| 元素 | 字号（近似值） | 字重 |
|------|-------------|------|
| Hero 大标题 | 40–60pt | 700 (Bold) |
| H1 | 28–36pt | 700 |
| H2 | 20–24pt | 600 |
| H3 | 16–18pt | 600 |
| 正文 | 14–16pt | 400 |
| 小字 / 标签 | 11–13pt | 400 |
| 表格 | 12–14pt | 400（表头 600） |

## 4. 版面与间距

| 参数 | 值 |
|------|------|
| 页面最大宽度 | 1200–1440px |
| 正文区宽度 | 700–900px |
| 页边距 | 宽松，多用 60–80px 两侧边距 |
| 段间距 | 宽松，明显大于正文行高 |
| 行距 | 1.5–1.65 倍 |

## 5. 组件风格

### 5.1 标题装饰

| 级别 | 装饰方式 |
|------|---------|
| Hero H1 | 黄色下划线 `#f6f600`，3–4px |
| H2 | 黄色左侧色条或无装饰 |
| H3 | 无装饰，字重 + 间距区分 |
| 引言文字 | 大字号 + Arnhem 衬线字体（如有授权） |

### 5.2 按钮

| 类型 | 样式 |
|------|------|
| 主要 CTA | 背景 `#ff3532` 红色，白色文字，无圆角或小圆角 |
| 次要 CTA | 背景 `#000`，白色文字；或边框按钮 `border: 1.5px solid #000` |
| 强调 CTA | 背景 `#f6f600` 黄色，黑色文字 |

### 5.3 表格（官网风格）

- **无竖线**，仅水平线区分层级
- 表头上方：粗黑线 `#000000`，2px
- 表头下方：品牌黄线 `#f6f600`，2px（区别于 McKinsey 黑色细线）
- 数据行之间：细灰分隔线 `#e0e0e0`，0.5px（McKinsey 无此行间线）
- 末行下方：黑色细线 `#000000`，1px
- 无斑马纹背景（纯白底）

### 5.4 卡片 / 区块

- 背景：白色 `#fff` 或深色 `#003b46`（深背景配白文字）
- 黄色 `#f6f600` 作为左侧装饰条（3–4px）区分内容类型
- Cyan `#00aac9` 用于图标色、标签底色

## 6. 格式推导指南

读取本文件后，按以下规则生成目标格式配置（在无法使用专有字体的环境下）：

### → 文档 / 报告（DOCX / PDF）

- 主色方案：**黑白为底，深青绿 `#003b46` 作文档主强调色，黄色仅小面积点缀**
- 标题颜色：`#003b46`（深青绿），H1/H2 带 `#003b46` 下划线
- H3 左侧色条：`#003b46`
- 表格表头下划线：`#f6f600`（黄色，唯一黄色出现位置）
- 引用块左侧装饰线：`#003b46`
- 代码块左侧装饰线：`#003b46`
- 字体降级栈：见 3.2 节

### → Mermaid 图表

- subgraph 颜色方案：黑底 `#000` 或深色 `#003b46`；白色文字
- 节点强调色：`#f6f600`（Yellow, 黑色文字）、`#00aac9`（Cyan, 黑色文字）、`#ff3532`（Red/警告, 白色文字）
- 数据流图：Cyan `#00aac9` 为主节点色，Yellow `#f6f600` 为关键路径节点
- 深色背景节点加 `color:#fff`，Yellow/Cyan 浅色节点用 `color:#000`
